## Python Utility Library

#### Installation
```bash
git clone https://github.com/karanveersp/python-utilfuncs
cd python-utilfuncs
pip install --editable .
```

#### Run tests
```bash
cd python-utilfuncs
pytest
```

Import the library:
    `import utilfuncs as util`
    
Use the library:
    `util.get_rows(csvpath, skip_header=True)`
